# Senior Data Scientist application

- The `R/final_submission.Rmd` file contains the RMarkdown file used to generate the overall report for the responses to the skills assessment.
- The `output/final_submission.html` file contains the formatted html document generated from the `final_submission.Rmd` file

- The `R/report.Rmd` file contains the RMarkdown file used to generate the deaths comparison report.
- The `output/deaths_qld_australia.html` file contains the formatted html document generated from the `report.Rmd` file
